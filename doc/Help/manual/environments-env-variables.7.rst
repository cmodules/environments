environments-env-variables(7)
**********************

.. only:: html

   .. contents::

This page lists environment variables that are accessible through the Environments api by default.

These can all be extended, over-rided, and further built-upon by the user.

For general information on environment variables, see the
:ref:`Environment Variables <CMake Language Environment Variables>`
section in the cmake-language manual.
